## 📌 Descrição

Bem-vindo ao meu repositório de estudos da **Trilha Iniciante em Programação G9** do programa **Oracle Next Education (ONE)**, uma incrível iniciativa de formação em tecnologia em parceria com a **Alura**, focada em preparar novos talentos para o mercado de trabalho.

Aqui, você encontrará os projetos e exercícios que desenvolvi, aplicando os fundamentos do desenvolvimento web com **HTML, CSS e JavaScript**. Este espaço serve como um portfólio do meu progresso e da minha dedicação em construir uma base sólida como desenvolvedor.

## 🚀 Projeto em Destaque: Jogo do Número Secreto

Como parte da formação, desenvolvi uma aplicação web completa para colocar em prática os conceitos aprendidos.

<h3 align="center">🎮 Jogo do Número Secreto</h3>

<p align="center">
  <a href="https://numero-secreto-orpin-five.vercel.app/" target="_blank">
    <img src="https://i.postimg.cc/zv9KPMjx/image.png" alt="Tela Inicial do Jogo" width="70%">
  </a>
</p>

O **Jogo do Número Secreto** é uma aplicação web interativa onde o jogador deve adivinhar um número secreto sorteado entre 1 e 10. O projeto foi fundamental para aplicar conceitos de lógica de programação, manipulação do DOM e interação com o usuário.

- **Funcionalidades Principais:**

  - Sorteio de números aleatórios.
  - Validação de tentativas com feedback visual e por voz.
  - Contador de tentativas.
  - Opção para reiniciar o jogo com um novo número secreto.

- **Destaques Técnicos:**
  - Manipulação dinâmica do DOM com JavaScript para atualizar textos e estilos.
  - Código estruturado em funções reutilizáveis.
  - Integração da API `responsiveVoice.js` para dar feedback por voz, melhorando a acessibilidade.
